Title: BackBox Linux 1 Final Release

<img class="alignleft" src="http://www.backbox.org/public/sites/default/files/corolla_logo.png" alt="" width="98" height="98" />Di <strong>BackBox</strong>, la distribuzione GNU/Linux orientata alla sicurezza vi avevo già parlato in <a href="http://www.evilsocket.net/1985/backbox-ottima-distribuzione-gnulinux-orientata-alla-sicurezza.html" target="_blank">questo</a> post risalente a circa un mese fa, periodo nel quale sono entrato a far parte del progetto inizialmente come core developer, successivamente (causa mancanza di tempo per lavoro :'() come semplice contributor.

Sono fiero, anche se di mio c'è poco, di annunciare che a distanza di 30 giorni dalla seconda release candidate, è pronta la versione 1.0 ufficiale!

Il <em>changelog</em> delle principali novità della release finale:

<break>
<ul>
	<li>New ISO image (32bit &amp; 64bit)</li>
	<li>System upgrade</li>
	<li>Performance boost (using <strong>cgroup</strong>, the notorious “<strong>4 lines of code better than 200</strong>″ in user-space)</li>
	<li>New look and feel</li>
	<li>Improved start menu</li>
	<li>Hacking tools new or updated: Aircrack-ng 1.1, Metasploit Framework 3.5.1, OpenVAS 3, SET 1.1, SkipFish 1.84b, SqlMap 0.9, w3af 1.0-rc4, etc.</li>
	<li>Bug fixing</li>
</ul>
