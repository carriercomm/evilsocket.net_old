Title: BackBox Linux 2 Released!

<p align="justify">
The BackBox team is proud to announce the release of BackBox Linux 2.

BackBox 2 features the following upstream components:

<ul>
  <li>Ubuntu 11.04</li> 
  <li>Linux Kernel 2.6.38</li> 
  <li>XFCE 4.8.0</li>
</ul>
<break>
<h4>What's new</h4>

<ul>
  <li>System upgrade</li>
  <li>Performance boost</li>
  <li>New look and feel</li>
  <li>Improved start menu</li>
  <li>Bug fixing</li>
  <li>Hacking tools new or updated</li>
  <li>Three new section: Vulnerabilty Assessment, Forensic Analysis and VoIP Analysis</li>
  <li>Much, much more!</li>
</ul>
